专家告诉你返程复工高峰期如何做好防护

发表于 2020-02-02 更新于 2020-02-05

这两天，很多人已经陆续开始返回工作和居住的城市，准备开工了。在返乡途中以及在工作场所怎么能保护好自己，避免病毒感染，是很多人关心的问题。让我们有请《医学通识50讲》的主理人薄世宁老师（薄老师是北京大学第三医院重症医学科的副主任医师，在2003年的时候曾经被评选为北京大学“抗击非典勇士”。）从专业的角度来聊一聊，在不同场景下采取防护措施，有哪些注意事项。

问题1：很多人已经开始返程复工了，在旅途中有哪些注意事项呢？

回答：首先，在返程的前几天，就要开始采取防护措施了。要避免过度劳累，适度运动，保证睡眠。尤其是每天保证七八个小时的睡眠时间，不要熬夜。睡觉可以增强T细胞反应的效率，维护好自己的免疫系统，让它保持在比较好的状态。
其次，在返程的途中还要注意以下7点：

  1. 全程佩戴口罩减少接触病原风险。你可以选择医用外科口罩、N95口罩。最好每2至4小时更换一次，避免反复使用，避免用手接触口罩外面。
  2. 一定不用脏手触摸眼睛和口鼻。不碰脸是最好的，避免交叉传染。
  3. 在路上减少与陌生人亲密交谈。这也是避免交叉传染的一种重要途径。
  4. 减少随身物品与外界的接触，比如书、手机、水杯等。如果发生接触，最好用消毒纸巾擦拭，而且要让消毒液自然风干，不要用手或纸巾抹干。可以自带一支笔，很多行程都会要求填写旅行登记卡。
  5. 如果不超过6个小时的路程，不建议在公共交通工具内吃饭。建议自带矿泉水，一次性喝完。
  6. 邻座如果发现频繁咳嗽，精神萎靡，请主动向工作人员汇报。主动汇报不仅保护你，也保护了邻座的人。
  7. 返程进家后，要把衣服向内裹起来塞进洗衣机。自己洗个热水澡，把头发洗干净，然后再收拾行李。

问题2：从这周开始有不少人需要回归工作岗位了，难免要在工作场所接触到大量的同事，以及在密闭的空间里办公、开会。怎么能在办公场所保护好自己？

回答：如果必须要回到工作岗位，对个人来说应该注意以下3点：

  1. 上班途中乘坐交通工具时，需要正确戴口罩，不用手碰眼睛和口鼻。尽量不和别人交谈、握手。到室内后要勤洗手，口罩用过一次后建议废弃不再二次应用。
  2. 督促每个同事养成经常洗手的好习惯，注意个人卫生。咳嗽或打喷嚏时，用纸巾或屈肘遮住口鼻。
  3. 如果你是在办公室一天上班8小时，长期戴口罩会造成缺氧、头晕等不舒适的感觉。如果确实办公室通风差，可以每一个小时适度到开阔地方稍作休息，运动，呼吸新鲜空气。

对于用人单位来说，需要注意以下5点：

  1. 在复工前应组织进行一次排查。重点排查有无武汉旅行史，体温是否正常以及有无接触可疑或者确诊的病例。
  2. 确保工作环境清洁卫生，保持室内空气流通。使用空调系统的单位，要定期清洗消毒空调系统。每天开启门窗，通风换气。
  3. 定期用消毒水对办公场所、台面、桌面、门把手和电梯按钮进行消毒。可以使用含氯消毒剂，比如84消毒液，使用浓度500mg/L。
  4. 减少不必要的会议、聚餐等群体性活动。
  5. 有同事出现发热、咳嗽等呼吸道症状，应劝其不上班，并尽早到正规医院进行就诊治疗。

问答3：怎么判断自己是否被感染？什么情况应该去医院，什么情况可以在家观察？ 根据媒体报道，目前已经出现了“无症状的感染者”，在这种可能性下，判断的标准会发生变化吗？

回答：你可以根据目前国家卫健委最新版的新型冠状病毒肺炎诊疗方案，来作出判断。

根据国家卫健委第四版新型冠状病毒肺炎诊疗方案，会通过流行病学史（接触史），以及现在是否符合临床表现症状这两类途径，来判断你是否属于疑似病例。

1. 流行病学史

1） 发病前14天内有到武汉地区，或其他有本地病例持续传播地区的旅行史或居住史。
2） 接触过来自于以上这些地区、有发热或者呼吸道症状的患者。
3） 跟聚集性发病的情况或与新型冠状病毒感染者有接触史。所谓的“聚集性病例”指的是14天内在小范围，比如一个家庭、一个工地、一个单位等，发现2例及以上确诊病例。

2. 临床表现

（1） 发热。
（2） 具有典型的肺炎影像学特征。
（3） 发病早期白细胞总数正常或降低，或淋巴细胞计数减少。

看这个指标是因为一般来说，应对细菌性的肺炎，白细胞会增高，这表示你的身体在进行防御。但是在新型冠状病毒的感染案例中，病人的化验检查却是白细胞正常或者减少。淋巴细胞减少，又意味着你的抗病毒部队被削弱了。
到底在什么情况下需要去医院检查，而什么情况下可以在家自己隔离呢？
如果你有流行病学史中的任何一条，并且符合临床表现中任意2条，那就基本可以判断，你属于疑似病例。
怎么就是确诊病例了呢？
在疑似病例的基础上，如果经过检测，呈现新型冠状病毒核酸阳性，或者经过病毒基因测序，与已知的新型冠状病毒高度同源，那么才能最终确诊。
所以，我再解释一下，如果你只是单纯的发热，并没有流行病学史，那么你可以在家居家隔离观察。
在你居家观察期间，应该注意单独居住在一个房间，通风良好。你尽量不和别人发生接触，并且要戴好口罩，规律作息，避免劳累，保持心态平和。
如果出现发热，可以应用退热药物。但是，一旦出现呼吸道症状，呼吸困难和干咳，就是没有痰的咳嗽，你就应该立即停止居家隔离，到医疗机构进行排查和就诊。
现在确实有了无症状感染者传染给其他人的病例报道，作为医生我要提醒你，这类病例只是让流行病学史变得更加复杂，患者的基本症状可能并不典型，增加了防控的难度。
但是患者的客观检查比如：抽血化验、胸部CT、核酸检测仍可发现问题。所以如果你有明确的接触史，又没有出现症状，这个时候做好自我隔离非常重要，以免把疾病传给周围的人。一旦出现症状，及时到医院就诊，筛查。
即便没有明显的发热和呼吸道症状，但是根据流行病学史+客观的检查指标+核酸检测阳性，仍然考虑为新型冠状病毒感染。

问题4：对一直待在家里不怎么出门的人来说，平时有什么注意事项吗？

回答：待在家中不出门，应该比较安全。我建议你，近期不去任何地方，避免近距离接触咳嗽、发热病人，并且尽量避免到医院探望病人。

对于待在家里的人，我的建议是：

  1. 正常生活，不要过度恐慌。多从官方渠道了解这一疾病流行的特点和发病情况，不要登录各种不靠谱的网站、自媒体公众号等，以免增加恐惧。像紧张、不安、恐惧这样的心理状态，会通过植物神经系统和内分泌系统影响我们的免疫力。也可能会引起一些身体的不适，比如说肠道功能紊乱、血压波动等等，这会进一步削弱免疫力。
  2. 保持家居清洁卫生，定期开窗保持空气流通。每天尽量开窗通风2次，每次至少30分钟。
  3. 保持每天7-8小时的良好睡眠。不要过度劳累，不要过度焦虑，恐惧，避免过度饮酒。
  4. 避免前往人群密集的场所。外出时最好佩戴医用外科口罩，4小时要更换一次。
  5. 注意个人卫生。咳嗽或打喷嚏时，用纸巾或屈肘遮住口鼻，勤洗手。
  6. 正常饮食，食物（尤其是肉和蛋类）要煮熟煮透。不接触和食用野生动物，多饮水。
  7. 身体不适及时到正规医院就医。如实告知医生自己在14天内的出行史。

以上就是薄老师的建议，希望你保护好自己和家人，在返工后尽快进入工作状态。

新型冠状病毒感染的肺炎防控知识手册

发表于 2020-01-29 更新于 2020-02-04

一、工作区域防护知识

1、上班途中如何做

正确佩戴一次性医用口罩。尽量不乘坐公共交通工具，建议步行、骑行或乘坐私家车、班车上班。如必须乘坐公共交通工具时，务必全程佩戴口罩。途中尽量避免用手触摸车上物品。

2、入楼工作如何做

进入办公楼前自觉接受体温检测，体温正常可入楼工作，并到卫生间洗手。若体温超过37.2℃，请勿入楼工作，并回家观察休息，必要时到医院就诊。

3、入室办公如何做

保持办公区环境清洁，建议每日通风3次，每次20-30分钟，通风时注意保暖。人与人之间保持1米以上距离，多人办公时佩戴口罩。保持勤洗手、多饮水，坚持在进食前、如厕后按照六步法严格洗手。接待外来人员双方佩戴口罩。

4、参加会议如何做

建议佩戴口罩，进入会议室前洗手消毒。开会人员间隔1米以上。减少集中开会，控制会议时间，会议时间过长时，开窗通风1次。会议结束后场地、家具须进行消毒。茶具用品建议开水浸泡消毒。

5、食堂进餐如何做

采用分餐进食，避免人员密集。餐厅每日消毒1次，餐桌椅使用后进行消毒。餐具用品须高温消毒。操作间保持清洁干燥，严禁生食和熟食用品混用，避免肉类生食。建议营养配餐，清淡适口。

6、下班路上如何做

洗手后佩戴一次性医用口罩外出，回到家中摘掉口罩后首先洗手消毒。手机和钥匙使用消毒湿巾或75%酒精擦拭。居室保持通风和卫生清洁，避免多人聚会。

7、公务采购如何做

须佩戴口罩出行，避开密集人群。与人接触保持1米以上距离，避免在公共场所长时间停留。

8、工间运动如何做

建议适当、适度活动，保证身体状况良好。避免过度、过量运动，造成身体免疫能力下降。

9、公共区域如何做

每日须对门厅、楼道、会议室、电梯、楼梯、卫生间等公共部位进行消毒，尽量使用喷雾消毒。每个区域使用的保洁用具要分开，避免混用。

10、公务出行如何做

专车内部及门把手建议每日用75%酒精擦拭1次。乘坐班车须佩戴口罩，建议班车在使用后用75%酒精对车内及门把手擦拭消毒。

11、后勤人员如何做

服务人员、安保人员、清洁人员工作时须佩戴口罩，并与人保持安全距离。食堂采购人员或供货人员须佩戴口罩和一次性橡胶手套，避免直接手触肉禽类生鲜材料，摘手套后及时洗手消毒。保洁人员工作时须佩戴一次性橡胶手套，工作结束后洗手消毒。安保人员须佩戴口罩工作，并认真询问和登记外来人员状况，发现异常情况及时报告。

12、公务来访如何做

须佩戴口罩。进入办公楼前首先进行体温检测，并介绍有无湖北接触史和发热、咳嗽、呼吸不畅等症状。无上述情况，且体温在37.2°正常条件下，方可入楼公干。

13、传阅文件如何做

传递纸质文件前后均需洗手，传阅文件时佩戴口罩。

14、电话消毒如何做

建议座机电话每日75%酒精擦拭两次，如果使用频繁可增加至四次。

15、空调消毒如何做

（1）中央空调系统风机盘管正常使用时，定期对送风口、回风口进行消毒。
（2）中央空调新风系统正常使用时，若出现疫情，不要停止风机运行，应在人员撤离后，对排风支管封闭，运行一段时间后关断新风排风系统，同时进行消毒。
（3）带回风的全空气系统，应把回风完全封闭，保证系统全新风运行。

16、废弃口罩处理如何做

防疫期间，摘口罩前后做好手卫生，废弃口罩放入垃圾桶内，每天两次使用75%酒精或含氯消毒剂对垃圾桶进行消毒处理。

什么是新型冠状病毒感染的肺炎？

发表于 2020-01-26 更新于 2020-02-06

概述

  • 新型冠状病毒（2019-nCoV）感染所致肺炎
  • 起初发热、乏力、干咳，逐渐出现呼吸困难等
  • 具备人传染人的能力，正确佩戴口罩加强防护
  • 多数预后良好，少数病情危重，甚至可致死亡

疾病定义

新型冠状病毒感染的肺炎是一种急性感染性肺炎，其病原体是一种先前未在人类中发现的新型冠状病毒，即2019新型冠状病毒（2019 novel Coronavirus，2019-nCoV）。2020年1月30日，世界卫生组织（WHO）建议将新型冠状病毒感染的肺炎命名为“2019-nCoV acute respiratory disease”。

2020年1月30日，WHO宣布将新型冠状病毒感染肺炎疫情列为国际关注的突发公共卫生事件（PHEIC）。

患者初始症状多为发热、乏力和干咳，并逐渐出现呼吸困难等严重表现。多数患者预后良好，部分严重病例可出现急性呼吸窘迫综合征或脓毒症休克，甚至死亡。目前，缺乏针对病原体的有效抗病毒药物，以隔离治疗、对症支持治疗为主。

流行病学

2020年1月30日，WHO宣布将新型冠状病毒疫情列为国际关注的突发公共卫生事件。

基于目前的流行病学调查，该病潜伏期一般为3~7天，最短的潜伏期为一天发病，最长的潜伏期是14天。潜伏期具有传染性。该病主要通过飞沫传播和接触传播，密闭、不通风场所可能存在气溶胶传播风险，需加强预防和隔离处理。人群普遍易感、老年人及有基础疾病者感染后病情较重，儿童及婴幼儿也有发病。

新型冠状病毒感染的肺炎首例患者于2019年12月12日在湖北省武汉市被发现，随后的患者多有华南海鲜市场暴露史或家庭聚集现象，目前感染源仍不明确，可能的传播途径有飞沫传播和接触传播。

患者最常见的问题

问：人感染新型冠状病毒后会有什么症状？
答：人感染新型冠状病毒后的症状严重程度取决于病毒的类型以及人体的免疫水平，常见的有发热、咳嗽、呼吸急促或呼吸困难，更严重时会导致急性呼吸窘迫综合征、脓毒性休克等，可导致患者死亡。

问：新型冠状病毒可以在人与人之间传播吗？
答：是的，新型冠状病毒是可以在人与人之间传播的，传播通常发生在在家庭、工作场所、人群密集区域或医院等。

问：目前有针对新型冠状病毒的疫苗吗？
答：没有，新发疾病可能需要很多年时间才能研发出接种的疫苗。

问：有针对新型冠状病毒的治疗方法吗？
答：对于由新型冠状病毒感染引起的疾病，目前没有特效的治疗方法，临床治疗以对症支持治疗为主。

问：医护人员会受到新型冠状病毒的威胁吗？
答：是的。医护人员需要与患者密切接触，更容易受到新型冠状病毒的感染威胁。因此，世界卫生组织（WHO）建议医护人员加强自身的感染预防和控制措施。严格执行标准预防原则，做好个人防护、手卫生，最大可能避免医院感染。

问：我该怎么做才能保护自己？
答：由于该病目前尚无有效治疗方法，预防与隔离是最有效途径，尽量减少与感染或潜在感染患者接触的机会。首先要避免前往人群密集场所，在公共场所要戴口罩。其次要注意手卫生和饮食卫生，勤洗手、多饮水、避免疲劳、保证睡眠，家庭和工作环境保持多通风。如果出现发热、咳嗽症状，要注意咳嗽礼仪，要及时就医，前往医院途中，戴口罩。
