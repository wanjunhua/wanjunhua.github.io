什么是新型冠状病毒感染的肺炎？

发表于 2020-01-26 更新于 2020-02-02

概述

  • 新型冠状病毒（2019-nCoV）感染所致肺炎
  • 起初发热、乏力、干咳，逐渐出现呼吸困难等
  • 具备人传染人的能力，正确佩戴口罩加强防护
  • 多数预后良好，少数病情危重，甚至可致死亡

疾病定义

新型冠状病毒感染的肺炎是一种急性感染性肺炎，其病原体是一种先前未在人类中发现的新型冠状病毒，即2019新型冠状病毒（2019 novel Coronavirus，2019-nCoV）。2020年1月30日，世界卫生组织（WHO）建议将新型冠状病毒感染的肺炎命名为“2019-nCoV acute respiratory disease”。

2020年1月30日，WHO宣布将新型冠状病毒感染肺炎疫情列为国际关注的突发公共卫生事件（PHEIC）。

患者初始症状多为发热、乏力和干咳，并逐渐出现呼吸困难等严重表现。多数患者预后良好，部分严重病例可出现急性呼吸窘迫综合征或脓毒症休克，甚至死亡。目前，缺乏针对病原体的有效抗病毒药物，以隔离治疗、对症支持治疗为主。

欢迎来到万俊华的个人网站！

发表于 2020-01-01 更新于 2020-02-01

你好！欢迎来到我的个人网站，我是万俊华博客的站长万俊华。

因为爱好互联网技术，所以有了这个网站。

在这里我会分享与我有关的一切——包括但不限于我的学习情况、工作情况以及生活情况。

当然，这一切都是经过我精心筛选过的正能量的东西和干货，我想把最好的一面展现给大家。

万俊华是谁？

万俊华，男，大学本科毕业于南昌大学共青学院，电子信息工程专业。

毕业后曾在幽兰司法所工作2年，目前在南昌县司法局富山司法所从事社区矫正工作。

为什么要建立本网站？

因为从小爱好计算机技术，在大学期间就因为有一门C语言课程，焕发了高考之后学习的第二春。

在大学期间接触并自学了 Linux 操作系统，也了解了 Linux 历史和文化。

工作之后，为了进一步培养兴趣爱好，购买了服务器和域名，因此才有了你看到的这个网站。

都说兴趣是最好的老师，因为兴趣，我可以废寝忘食的持续完善我的个人网站，让这个网站变得更好是我的一大动力。

这个网站是有独立服务器和独立域名的，我可以充分发挥特长，更好的展示自我风采，同时也能让关心和关爱我的亲朋好友更好的了解我，以及了解我所从事的工作。

活到老，学到老！我相信这个网站经过慢慢的积累，时间可以赋予她持续不断的力量，最后让所有浏览她的人受益。
